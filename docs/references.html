Pacotes R Utilizados

Este livro faz uso extensivo de diversos pacotes do ecossistema R. Agradecemos aos desenvolvedores e mantenedores dessas ferramentas:

Manipulação e Visualização de Dados

  • tidyverse [@wickham2019]: Coleção de pacotes para ciência de dados
    • Wickham H, Averick M, Bryan J, et al. (2019). “Welcome to the tidyverse.” Journal of Open Source Software, 4(43), 1686. https://doi.org/10.21105/joss.01686
  • dplyr [@wickham2023dplyr]: Manipulação de dados
    • Wickham H, François R, Henry L, Müller K (2023). dplyr: A Grammar of Data Manipulation. https://dplyr.tidyverse.org
  • ggplot2 [@wickham2016]: Visualização de dados
    • Wickham H (2016). ggplot2: Elegant Graphics for Data Analysis. Springer-Verlag New York. https://ggplot2.tidyverse.org
  • readxl [@wickham2023readxl]: Leitura de arquivos Excel
    • Wickham H, Bryan J (2023). readxl: Read Excel Files. https://readxl.tidyverse.org

Estatística e Modelagem

  • gtsummary [@sjoberg2021]: Tabelas de resumo estatístico
    • Sjoberg DD, Whiting K, Curry M, Lavery JA, Larmarange J (2021). “Reproducible Summary Tables with the gtsummary Package.” The R Journal, 13(1), 570-580. https://doi.org/10.32614/RJ-2021-053
  • jtools [@long2020]: Análise e visualização de modelos de regressão
    • Long JA (2020). jtools: Analysis and Presentation of Social Scientific Data. https://jtools.jacob-long.com
  • car [@fox2019]: Funções complementares para análise de regressão
    • Fox J, Weisberg S (2019). An R Companion to Applied Regression, Third edition. Sage, Thousand Oaks CA. https://socialsciences.mcmaster.ca/jfox/Books/Companion/
  • lmtest [@zeileis2002]: Testes para modelos lineares
    • Zeileis A, Hothorn T (2002). “Diagnostic Checking in Regression Relationships.” R News, 2(3), 7-10. https://CRAN.R-project.org/package=lmtest
  • Batra, Neale, et al. The Epidemiologist R Handbook. 2021. https://www.epirhandbook.com/en/

Tabelas e Relatórios

  • flextable [@gohel2023]: Criação de tabelas para publicação
    • Gohel D, Skintzos P (2023). flextable: Functions for Tabular Reporting. https://ardata-fr.github.io/flextable-book/
  • officer [@gohel2023officer]: Manipulação de documentos Office
    • Gohel D (2023). officer: Manipulation of Microsoft Word and PowerPoint Documents. https://davidgohel.github.io/officer/

Utilidades

  • janitor [@firke2023]: Limpeza de dados
    • Firke S (2023). janitor: Simple Tools for Examining and Cleaning Dirty Data. https://sfirke.github.io/janitor/
